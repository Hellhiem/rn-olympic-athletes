## Installation

**NOTE**:  If you never ran a React Native project, go to
  https://reactnative.dev/docs/environment-setup setup enviroment. 

**NOTE**:  Remember to run `athletes-backend`. 

### Install packages
  ```bash
    yarn
  ```
### Install pods
  ```bash
    cd ios && pod install
  ```


## How to run app

### iOS
```bash
  react-native run-ios
```

### Android
```bash
  react-native run-android
```
